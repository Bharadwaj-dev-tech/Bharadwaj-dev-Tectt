## â“ **FAQ**

<details>
<summary><b>Is web scraping legal?</b></summary>

Web scraping exists in a legal gray area. Always:
1. Check the website's Terms of Service
2. Review robots.txt
3. Be mindful of data protection laws (GDPR, CCPA)
4. Use publicly available data only
5. Consider consulting legal advice for commercial projects
</details>

<details>
<summary><b>How do I avoid getting blocked?</b></summary>

- Implement delays between requests
- Rotate user agents and IP addresses
- Respect `robots.txt`
- Use headless browsers sparingly
- Monitor for CAPTCHAs and adjust behavior
</details>

<details>
<summary><b>What's the difference between scraping and crawling?</b></summary>

**Crawling** discovers and indexes web pages (like search engines)
**Scraping** extracts specific data from web pages

This repository covers both techniques!
</details>
